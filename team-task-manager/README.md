Team Task Manager 

A full-stack web application to manage projects and tasks.

Features

- User Signup/Login
- Create Projects
- Delete Projects
- Dashboard View
- MongoDB Database

Tech Stack

- Frontend: React
- Backend: Node.js + Express
- Database: MongoDB

How to Run

Backend

###cd backend
npm install
node server.js

###Frontend

cd frontend
npm install
npm run dev
